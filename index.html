AppleScript

Apple's built-in scripting language for macOS automation. Uses native dialogs, progress bars, and system integration that ship with every Mac. Apple Documentation

⚙️

Automator & Shortcuts

Part of Apple's automation ecosystem. Build workflows visually with Automator or the Shortcuts app on macOS Monterey and later.

🛠️

osacompile

Apple's compiler that builds AppleScript into native macOS .app bundles. Double-click to run like any native application. Apple Developer

Toolbox is maintained by CodeCraftedApps, an independent developer building open-source tools for macOS. Every tool uses Apple's native automation technologies — AppleScript, Automator, and Shortcuts — to solve real problems without third-party frameworks or runtimes.
