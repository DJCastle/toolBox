How It Works

Every tool is open source. Clone the repo and run the scripts — or use one-click launchers.

🔍

Browse & Choose

Find the tool you need from the collection. Each one solves a specific automation problem — from dev environment setup to batch downloads and repo management.

💻

Clone & Run

Every tool lives in one GitHub repository. Clone it, run the script for your platform — shell on macOS, PowerShell on Windows. No installers or runtimes required.

🚀

One-Click Launchers

Prefer not to use the terminal? Import an Apple Shortcut (macOS) or Power Automate Desktop flow (Windows) for one-click execution.

Built With Native Automation Technologies

No third-party frameworks or runtimes required. Uses what ships with your OS.

📝

Shell Script & AppleScript

macOS tools use Bash shell scripts and Apple's built-in AppleScript for native dialogs and system integration. Package management via Homebrew.

⚙️

PowerShell & Chocolatey

Windows tools use PowerShell scripts. Package management via Chocolatey. No external dependencies beyond what Windows provides.

🚀

Shortcuts & Power Automate

One-click launchers for both platforms. Apple Shortcuts on macOS Monterey+, Power Automate Desktop on Windows 10/11.

About

Toolbox is maintained by CodeCraftedApps, an independent developer building open-source tools for macOS and Windows. Every tool uses native automation technologies — shell scripts, PowerShell, AppleScript, Shortcuts, and Power Automate Desktop — to solve real problems without third-party frameworks.
