# Apple Shortcuts

Automation workflows for Apple's [Shortcuts.app](https://support.apple.com/guide/shortcuts-mac/intro-to-shortcuts-apdf22b0444c/mac). Import and run with one click — no coding required.

---

## Security

These shortcuts run shell scripts. Before using them:

1. Open **Shortcuts.app** > **Settings** > **Advanced**
2. Enable **Allow Running Scripts**
3. Run the shortcut
4. **Disable "Allow Running Scripts" when done** — best practice to keep this off when not in use

---

## Available Shortcuts

| Shortcut | Description |
| -------- | ----------- |
| [non-appstore-download](non-appstore-download/) | Batch download non-App Store apps to your Desktop |
| [package-manager-setup](package-manager-setup/) | Set up a macOS dev environment with Homebrew |
| [repo-sync](repo-sync/) | Clone and update all your GitHub repos |
| [app-downloader](app-downloader/) | Download app installers to your Downloads folder |
