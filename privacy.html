Privacy Policy

Last updated: February 18, 2026

Toolbox is an open-source project by CodeCraftedApps ("we," "our," or "us"). This Privacy Policy explains how data is handled when you use this tool, visit this website, or interact with our GitHub repository.

Our Privacy Commitment

This tool is designed with a privacy-first approach. It runs entirely on your local machine and is built to respect your privacy by design.

Data the Tool Collects

No Data Collection

Toolbox operates entirely on your local machine. The tool:

  • Does not transmit any data to servers owned by us
  • Does not collect analytics, telemetry, or usage statistics
  • Does not phone home or check for updates automatically
  • Does not require account creation or authentication

What Happens When You Run It

When you run the AppleScript application or shell script, it makes requests to the Homebrew Cask API to resolve download URLs, then downloads files from third-party vendor servers using curl. These requests go directly to the vendor's servers. We have no visibility into these requests.

Third-Party Services

The tool connects to the following services:

  • Homebrew Cask API: Used to resolve the latest stable release URL for each app. Subject to Homebrew's BSD License

The default app list downloads from these vendors (each with their own privacy policies):

  • Google: Used for Chrome downloads. Subject to Google's Privacy Policy
  • Brave: Subject to Brave's Privacy Policy
  • Microsoft: Used for VS Code downloads. Subject to Microsoft's Privacy Statement
  • Slack (Salesforce): Subject to Slack's Privacy Policy
  • iTerm2: Open-source terminal. Subject to iTerm2's website

Users can add any app available in Homebrew Cask. Downloads go directly to each vendor's servers — we do not proxy, cache, or modify any downloads.

Website Data

This Website

This website (toolbox.codecraftedapps.com) is a static site hosted on GitHub Pages. We do not use cookies, tracking pixels, or analytics services of our own on this site.

  • GitHub Pages may collect standard server logs (IP address, browser type, pages visited) as part of hosting. See GitHub's Privacy Statement for details
  • Google Fonts are loaded from Google's CDN, which is subject to Google's Privacy Policy
  • A Buy Me a Coffee donation widget is loaded on the homepage. This third-party script may collect usage data such as page views and interactions. See Buy Me a Coffee's Privacy Policy for details

Open-Source Transparency

As an open-source project, all code is publicly available for review on GitHub. You can inspect exactly what the tool does before running it. This transparency is a core part of our privacy commitment.

Children's Privacy

This tool is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date. Significant changes will also be reflected in the repository's commit history.

Parent Organization

Toolbox is a project of CodeCraftedApps. For the general privacy policy covering all CodeCraftedApps projects, please see the CodeCraftedApps Privacy Policy. Where this tool-specific policy provides more detail, it takes precedence.

Contact

If you have questions about this Privacy Policy, please reach out through the CodeCraftedApps Contact page or open an issue on GitHub.
